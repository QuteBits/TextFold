TextFold 0.1
========

Fold text, a single JS script. Makes all &lt;span>-wrapped chunks of text foldable.

Currently only the Proof-of-Concept example. See the demo: [http://jsfiddle.net/FRp57/4/](http://jsfiddle.net/FRp57/4/)
